字體單位設定

參考文章：一次搞懂 CSS 字體單位：px、em、rem 和 %

（ 作者還有很多實用前端技術文章 ）

  • px : 絕對單位 固定 pixel
  • em : 相對單位 只跟根元素相乘 自己x祖先(或預設)
  • rem: 相對單位 與該父元素相乘 自己x老爸x老爸的老爸

-child

  • first-child：第一個子元素
  • last-child：最後一個子元素
  • nth-child(數字)：第幾個子元素
  • nth-child(2n)：挑出偶數的元素(2的倍數)
  • nth-child(2n+1)：挑出奇數的元素
  • nth-last-child(數字)：從後面數來第幾個子元素
  • only-child：只有一個子元素

p>span : <p> 中的第一個 <span>
